项目
====

### 克隆项目
- $`git clone git@github.com:johnnynode/react-native-test.git --depth 1`

### 全局安装yarn
- $ `npm i -g yarn`

### 安装项目
- $`npm i` 或 $`yarn install`

### 运行项目到手机
- 连接手机后执行, $`adb devices` 显示已连接的手机
- $`react-native run-android`

### iOS Debug

- cmd + R 刷新
- cmd + D 弹出功能窗

### Android Debug

- 双击 R
- cmd + M 弹出功能窗
